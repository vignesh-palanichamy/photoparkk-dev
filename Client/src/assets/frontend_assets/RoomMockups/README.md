# Room Mockup Images

## How to Add the Room Background Image

1. Download or save the room image you want to use
2. Save it as `room-landscape.png` in this directory
3. The image should be a PNG or JPG/JPEG file showing a room with a wall where frames can be displayed

## Current Image

- **File name**: `room-landscape.png`
- **Location**: `Client/src/assets/frontend_assets/RoomMockups/room-landscape.png`
- **Usage**: Used in AcrylicLandscapeOrderpage.jsx for the room preview

## Image Requirements

- Format: PNG (current) or JPG/JPEG
- Recommended size: 900x600px or larger (will be scaled to fit)
- Should show a room/wall where a picture frame would be displayed
